# FinSight AI

Персональный анализатор банковских расходов с приватной обработкой данных прямо в браузере.

## Возможности

- импорт выписок CSV, XLS и XLSX;
- объединение операций Сбера, Альфа-Банка и ВТБ;
- автоматическая категоризация и динамика расходов;
- поиск крупных и повторяющихся трат;
- персональные выводы и оценка потенциала экономии;
- локальные предложения для района Осташковской улицы в Москве;
- встроенная синтетическая демонстрационная выборка.

## Приватность

Файлы обрабатываются локально в браузере и не отправляются на сервер. Репозиторий не содержит реальных банковских данных.

## Запуск

Откройте `index.html` или воспользуйтесь опубликованной GitHub Pages-версией. Нажмите «Запустить демо», чтобы увидеть интерфейс без загрузки собственной выписки.

> Проект находится на стадии функционального прототипа. PDF-импорт и сравнение актуальных цен требуют дополнительных проверяемых источников данных.


## Локальный AI

Кнопка «Запустить локальный AI» загружает Qwen2.5-0.5B-Instruct через WebLLM и выполняет генерацию на GPU пользователя. API-ключ, платная подписка и сервер владельца не используются. В браузерах без WebGPU остаются доступны точные расчёты по правилам.

При первом запуске модель загружается из публичного хранилища и затем кэшируется браузером.

## Обучение и проверка

В `training/` находится воспроизводимый генератор 2 500 синтетических примеров для LoRA/QLoRA и независимой проверки. Реальные банковские данные не используются. Дообученный адаптер будет опубликован только после измеримого превосходства над базовой моделью по валидности JSON, точности чисел и отсутствию выдуманных фактов.
