# Подготовка локальной модели

Генератор создаёт 2 500 синтетических русскоязычных примеров: 2 000 для обучения, 250 для проверки и 250 для финального теста. Реальные банковские операции не используются.

Запуск: `python generate_dataset.py`.

Следующий исследовательский этап — LoRA/QLoRA и сравнение с базовой моделью на `test.jsonl`. Адаптер нельзя публиковать как улучшение, пока он не превосходит базовую модель по валидности JSON, точности чисел и отсутствию выдуманных фактов. Поэтому сайт сейчас использует браузерную Qwen2.5-0.5B-Instruct с доменной системной инструкцией.
